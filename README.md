# Detailed Website Description: Intimacy Wellness Industry - Facts & Figures

## 📋 **Overview**

"**Wellness Insights**" is a modern, dynamic, single-page website designed to showcase comprehensive facts and figures about the global intimacy wellness industry. Built with HTML, CSS, and JavaScript, the website combines elegant design with interactive content delivery to provide an engaging user experience for professionals, researchers, entrepreneurs, and the general public interested in this rapidly growing sector.

---

## 🎯 **Purpose & Target Audience**

The website serves as an informational hub that:
- **Educates** visitors about the expanding intimacy wellness market
- **Presents data-driven insights** backed by market research
- **Highlights industry trends** and consumer behaviors
- **Showcases growth opportunities** in this emerging sector
- **Provides regional and segment analysis** for market understanding

**Target Audience:**
- Industry professionals and entrepreneurs
- Market researchers and investors
- Healthcare and wellness practitioners
- Students and educators
- Curious consumers interested in market trends

## ⚙️ **Technical Architecture**

### **Files & Structure**
1. **index.html** - Semantic HTML5 structure with proper accessibility
2. **style.css** - 500+ lines of modern CSS with:
   - CSS Grid and Flexbox layouts
   - CSS animations and transitions
   - Media queries for responsiveness
   - CSS custom properties (variables)
3. **data.js** - Data module containing:
   - Facts array (8 items)
   - Statistics array (6 items)
   - Market segments, growth drivers, regional data, consumer trends
4. **script.js** - Dynamic functionality including:
   - DOM content loading
   - Dynamic content rendering
   - Event listeners for navigation
   - Intersection Observer for scroll animations

### **Key Technologies**
- **HTML5**: Semantic markup with proper structure
- **CSS3**: Modern styling with gradients, animations, and flexbox/grid
- **Vanilla JavaScript**: No frameworks, lightweight and fast
- **Responsive Web Design**: Mobile-first approach

### **Performance Features**
- **Lightweight**: Minimal file sizes
- **No external dependencies**: Pure HTML/CSS/JS
- **Fast loading**: Optimized code structure
- **Smooth animations**: GPU-accelerated transforms
